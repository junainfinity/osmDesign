# Project Modes

osmDesign starts new work from three creation paths:

- **Prototype** - build an interactive or responsive HTML prototype.
- **Slide deck** - build a deck with slide navigation and export-friendly print styles.
- **From template** - start from a saved project snapshot.

## Prototype

Prototype projects default to high-fidelity output unless the user chooses wireframe. The selected design system is injected as the visual source of truth when one is chosen.

## Slide Deck

Deck projects use the shared deck framework so keyboard navigation, slide count, print layout, and export behavior remain stable across model turns.

## From Template

Templates are saved from existing projects. Starting from a template copies the saved files into a new project and asks the model to continue from that base.

## Imports

The design ZIP importer accepts exported `.zip` files, unpacks safe project-relative paths, and opens the detected HTML entry file in the preview pane.
